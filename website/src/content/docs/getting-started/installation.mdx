## Requirements

- Go 1.26 or later (for building from source)
- Or use the pre-built Docker image via GitHub Actions

## Install the CLI

```bash
go install github.com/larsartmann/md-go-validator@latest
```

Verify the installation:

```bash
md-go-validator --version
```

## GitHub Action

Use the built-in GitHub Action in your workflows (no install needed):

```yaml
- uses: LarsArtmann/md-go-validator@v1
  with:
    path: .
    args: "-f json"
```

## As a Go Library

```bash
go get github.com/larsartmann/md-go-validator/pkg
```

Import in your code:

```go
import (
    mdgovalidator "github.com/larsartmann/md-go-validator/pkg"
    "github.com/larsartmann/md-go-validator/pkg/languages"
)
```

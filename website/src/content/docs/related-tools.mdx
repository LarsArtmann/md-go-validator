## Complementary Tools

md-go-validator works well alongside:

- **markdownlint** — use markdownlint for structure, md-go-validator for code syntax
- **golangci-lint** — use golangci-lint for Go source code, md-go-validator for Go in docs
- **pre-commit** — run both formatters and validators in your hooks
